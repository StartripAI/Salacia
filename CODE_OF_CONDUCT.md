# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Sharing private information without consent.
- Deliberate disruption of project collaboration.

## Enforcement

Project maintainers may remove comments, reject contributions, or ban participants for violations.

## Reporting

Report conduct issues privately to project maintainers.
